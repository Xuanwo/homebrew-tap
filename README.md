# Xuanwo Tap

## How do I install these formulae?

`brew install xuanwo/tap/<formula>`

Or `brew tap xuanwo/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "xuanwo/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
